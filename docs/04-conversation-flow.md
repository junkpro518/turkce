# 04 — دورة المحادثة (الذكاء vs الخلفية)

> المبدأ: **الردّ لا يُحجب أبداً بانتظار التحليل.** المحادثة فورية وذكية؛ التعلّم يجري صامتاً في الخلفية. هذا تجسيد طلب "ذكي كأني أكلم كلاود".

## دورة حياة الرسالة

```
1. الطالب يرسل رسالة (تيليجرام/ويب)
        ↓
2. بناء السياق (حتمي، سريع)
   المنسّق يقرأ: نموذج المتعلّم + هدف المنهج الحالي + أبرز الضعف
   + المفردات ذات الصلة + ملخص آخر جلسة  →  سياق متدرّج موفّر للتوكنات
        ↓
3. نداء المعلّم (ذكي، streaming)
   [prompt تربوي + حالتك الحيّة] + تاريخ المحادثة + رسالتك
   → المعلّم يقرّر النمط بحُكمه ويردّ طبيعياً (يُبثّ فوراً)
   → إن اختار الاختبار: يُخرج payload منظّماً (generateObject)
        ↓
   ┌─────────── الرد يصل للطالب الآن ───────────┐
        ↓
4. التحليل في الخلفية (لا يؤخّر الرد)
   نداء محلِّل منظّم (Zod) يراجع رسالة الطالب
   → أخطاء (تصنيف + شرح) + مفردات جديدة + إشارات إتقان
        ↓
5. التحديث في الخلفية (دوال نقية)
   تحديث grammar_mastery · upsert vocab_cards (FSRS) · إضافة error_log
   دورياً: تحديث تقدير CEFR + توليد ملخص الجلسة
```

## السياق المتدرّج (Tiered Context)
لتفادي prompt ضخم (الذي أبطأ النسخ السابقة):
- **Tier 1 (ثابت)**: المستوى، اللغة، تفضيلات التدريس.
- **Tier 2 (ديناميكي)**: هدف المنهج الحالي · آخر 5 أخطاء مختلفة · أضعف ~20 كلمة · آخر موضوع · ملخص آخر جلسة.
- يُبنى من نموذج المتعلّم — لا استعلامات زائدة.

## استراتيجية prompt المعلّم
- **هوية + منهجية**: معلّم تركي دافئ، يتحدّث العربية + التركية (لا إنجليزية)، يقرأ الطالب ويقرّر النمط.
- **حقن الحالة**: نبذة موجزة عنك (مستوى، هدف، ضعف، مفردات ذات صلة) — تجعل قراراته **مرتكزة عليك** لا عشوائية.
- **توجيه الأنماط**: وصف متى يناقش/يجيب/يختبر/يحكي/يمارس — لكن **القرار حُكمه**، لا قاعدة صلبة.
- **بروتوكول الاختبار**: حين يقرّر الاختبار، يُخرج بنية منظّمة (سؤال + خيارات + الصحيح + الشرح) تُعرَض كبطاقة.
- **الذاكرة**: ملخصات الجلسات السابقة تُحقن ليبني على ما سبق.

## فصل المسؤوليات (لماذا نداءان)
| النداء | الغرض | الخصائص |
|--------|-------|---------|
| **المعلّم** | المحادثة الحيّة | streaming، حرارة أعلى، حُكم تربوي |
| **المحلِّل** | تحليل رسالة الطالب | منظّم (Zod)، حرارة منخفضة، خلفي |

الفصل يضمن: محادثة ذكية لا تتعطّل، وتحليلاً دقيقاً قابلاً للاختبار. (الدمج في نداء واحد أو في graph معقّد = خطأ النسخ السابقة.)

## معالجة الفشل
- فشل المحلّل لا يكسر المحادثة (degradation سلس — التحليل اختياري للدور).
- فشل المعلّم ⇒ رسالة ودّية + إعادة محاولة/نموذج احتياطي.
- كل نداء AI محاط بـ try/catch + logging (انظر [07](07-quality-and-pitfalls.md)).
