# turkce — معلّم اللغة التركية بالذكاء الاصطناعي (إعادة بناء)

> مدرّب تركي شخصي بالـ AI، يتمحور حول **نموذج حيّ عنك** يتكيّف مع مستواك وأهدافك المتغيّرة.
> واجهة أساسية: **تيليجرام** · واجهة مرافقة: **ويب خفيف للوحات**.

هذا المجلد يحوي **خطة المشروع الكاملة** قبل أي كود. النسخة الجديدة إعادة بناء من الصفر، مبنية على دروس مستخلصة من نسختين فاشلتين سابقتين (`TR_AR` الويب، و`TR_AR_AI` التيليجرام).

## المبدأ الواحد الحاكم

> **الذكاء في المحادثة، والحتمية في الأنظمة المساندة.**
> المعلّم وكيل محادثة ذكي بحُكم تربوي (يقرّر متى يناقش/يجيب/يختبر/يحكي). أما نموذج المتعلّم وSRS وتتبّع الإتقان فأنظمة **حتمية مُختبَرة**. لا graph معقّد، ولا جسر فارغ يتّكئ على إطار خارجي.
